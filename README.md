# LWJGUI
An LWJGL3-based JavaFX alternative for making user-interfaces in Java. It can be incorperated into your already existing opengl project, or you can make your projects rendering become managed by LWJGUI.


# Current Features
- Button
- ComboBox
- CheckBox
- RadioButton
- SegmentedButton
- ContextMenu
- MenuBar
- ToolBar
- Label
- SplitPane
- Pane/StackPane
- BorderPane
- ScrollPane
- OpenGLPane
- TabPane
- BlurPane
- TextArea
- CodeArea
- HBox
- VBox
- Font
