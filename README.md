# LWJGUI
Simple library to add a user interface in LWJGL3.


# Current Features
- Button
- CheckBox
- RadioButton
- ContextMenu
- MenuBar
- Label
- SplitPane
- Pane/StackPane
- BorderPane
- ScrollPane
- OpenGLPane
- BlurPane
- TextArea
- CodeArea
- HBox
- VBox
- Font
