# LWJGUI
Simple library to add a user interface in LWJGL3.


# Current Features
- Button
- CheckBox
- RadioButton
- ContextMenu
- MenuBar
- Label
- Pane/StackPane
- SplitPane
- HBox
- VBox
- Font
