# LWJGUI
Simple library to add a user interface in LWJGL3.


# Current Features
- Button
- CheckBox
- RadioButton
- SegmentedButton
- ContextMenu
- MenuBar
- ToolBar
- Label
- SplitPane
- Pane/StackPane
- BorderPane
- ScrollPane
- OpenGLPane
- TabPane
- BlurPane
- TextArea
- CodeArea
- HBox
- VBox
- Font
